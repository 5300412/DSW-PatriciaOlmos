proposito

La creación de páginas web está conformada por contenidos accesibles que permiten incorporar material interactivo para crear formas novedosas en la construcción de ambientes digitales.

ETIQUETAS SEMANTICAS

Para crear las etiquetas semánticas, las cuales pertenecen al un estándar para facilitar interacción con la multimedia para la presentación del contenido Según Diez, T., Domínguez, M. J., Martínez, J. J., & Sáenz, J. (2012). Las etiquetas semánticas es una balise que describe las características de tipo y género textuales que se encarga de dar presentación a una página web. Por otro lado al utilizar CSS Grid este contenido es para establecer en cuadricula con ítems en línea que se pueden visualizar de forma instantánea.
